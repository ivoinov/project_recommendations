  <name>Task 1: Add App Runner deploy/verify/rollback scripts</name>
  <files>
    scripts/deploy/apprunner_deploy.sh
    scripts/deploy/apprunner_verify.sh
    scripts/deploy/apprunner_rollback.sh
  </files>
  <action>
    Create shell scripts to build/push API and worker images to ECR and update both App Runner services.
    - Use commit SHA as the image tag for both services.
    - Capture previous image identifiers for API and worker before updating (store in a temp file for rollback).
    - Update App Runner services via `aws apprunner update-service` for both API and worker.
    - Wait until both services reach `RUNNING` before verification.
    - Verify health via `curl` to `API_HEALTH_URL` and `WORKER_HEALTH_URL`.
    - On verification failure: rollback both services to previous image identifiers, retry verification once, then exit non-zero if still failing.
    - Include `set -euo pipefail` and clear logging output for CI.
  </action>
  <verify>
    bash -n scripts/deploy/apprunner_deploy.sh && \
    bash -n scripts/deploy/apprunner_verify.sh && \
    bash -n scripts/deploy/apprunner_rollback.sh
  </verify>
  <done>Deploy scripts exist, are executable, and implement update, wait, verify, and rollback flow.</done>
